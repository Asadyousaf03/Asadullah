FloorGen AI

This system takes natural language input and generates detailed floor plans using the FLUX-Schnell diffusion model. It consists of two core modules: an image generation module that creates realistic floor plan visuals, and a raster-to-vector conversion module that transforms these images into editable vector or labeled formats. An evaluation function measures layout quality, enabling users to choose and export the best design. The generated floor plans can be used in game development, architectural visualization, and AI-assisted design applications, providing a smart and efficient way to turn textual descriptions into structured, realistic floor layouts.

YTBot

PYTHON LangChain FIASS HTML CSS/JS

A Flask-based AI application that enables users to interact with YouTube videos through natural language. By providing a video link (with captions), the system fetches and embeds transcripts using FAISS and Hugging Face multilingual embeddings, allowing multilingual summarization and Q&A. Powered by a RAG architecture with Gemini (gemini-2.5-flash), it delivers accurate, context-aware answers while maintaining active sessions for seamless interaction. The system provides an intuitive way to explore and understand video content efficiently, making it ideal for educational, research, and content analysis purposes.

Story Generation Using LSTM and GRU

Vue.js Express Socket.io PostgreSQL

This project focuses on generating creative stories from short user-provided outlines using Recurrent Neural Networks (RNNs). Users input a 50-word summary, and the system expands it into a coherent, full-length story. The model was trained on large-scale textual data after extensive data scraping and cleaning to capture narrative flow and language patterns. Both LSTM and GRU architectures were implemented and compared in terms of story coherence, creativity, and training efficiency. Experimental results showed that GRU outperformed LSTM on short sequences due to its simpler structure and faster convergence, making it better suited for lightweight text generation tasks.
